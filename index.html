Whoop
›
1
Create developer account
Go to developer.whoop.com, create an app called "Baseline", and set the redirect URI to your Netlify URL + /api/whoop-callback.
2
Add credentials to Netlify
In Netlify → Site settings → Environment variables, add: WHOOP_CLIENT_ID, WHOOP_CLIENT_SECRET, WHOOP_REDIRECT_URI, SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY.
3
Connect below
After adding credentials, tap Connect to authorize Baseline to read your Whoop data.
